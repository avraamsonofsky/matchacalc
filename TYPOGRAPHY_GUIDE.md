# Руководство по типографике MatchaCalc

## Принципы типографики, применённые в дизайне

### Межбуквенные интервалы (Letter-spacing)

**Правило:** Чем больше размер шрифта, тем меньше (или даже отрицательное) letter-spacing нужно для визуальной плотности.

#### Заголовки (H1 - 90px)
- **Letter-spacing: -0.02em** (-1.8px)
- **Обоснование:** Для очень больших заголовков отрицательный tracking создаёт визуальное единство и элегантность
- **Практика:** Apple, Google, современные премиум-бренды используют отрицательный tracking для больших заголовков

#### Заголовки средние (H2 - 56px)
- **Letter-spacing: -0.01em** (-0.56px)
- **Обоснование:** Небольшое отрицательное значение для плотности, но не такое агрессивное как H1

#### Заголовки малые (H3 - 32px, H4 - 24px)
- **Letter-spacing: 0em - 0.01em**
- **Обоснование:** Нейтральное значение, шрифт сам по себе хорошо читается

#### Основной текст (Body - 16px)
- **Letter-spacing: 0.02em** (0.32px)
- **Обоснование:** Небольшое увеличение улучшает читаемость, особенно для длинных текстов
- **Практика:** Рекомендуется для body text в современном веб-дизайне

#### Мелкий текст (Small - 14px, XS - 12px)
- **Letter-spacing: 0.03em - 0.04em**
- **Обоснование:** Увеличенный tracking компенсирует маленький размер и улучшает читаемость

#### Логотипы и кнопки
- **Letter-spacing: 0.05em - 0.08em**
- **Обоснование:** Увеличенный tracking создаёт премиальный вид и улучшает визуальное восприятие

---

### Межстрочные интервалы (Line-height)

**Правило:** Чем больше размер шрифта, тем меньше line-height (для заголовков). Для body text - комфортное чтение.

#### Заголовки (H1 - 90px)
- **Line-height: 1.0** (очень плотный)
- **Обоснование:** Для больших заголовков плотный line-height создаёт визуальное единство
- **Практика:** Используется в современном минималистичном дизайне (Surf Coffee, Apple)

#### Заголовки средние (H2 - 56px)
- **Line-height: 1.1**
- **Обоснование:** Немного больше воздуха, но всё ещё плотный

#### Заголовки малые (H3 - 32px)
- **Line-height: 1.2**
- **Обоснование:** Умеренный spacing для читаемости

#### Основной текст (Body - 16px)
- **Line-height: 1.6**
- **Обоснование:** Золотая середина для комфортного чтения
- **Практика:** Рекомендуется W3C и большинством типографских руководств
- **Исследования:** Оптимально для чтения с экрана (1.5-1.8)

#### Мелкий текст
- **Line-height: 1.5**
- **Обоснование:** Немного меньше, так как строки короче

---

## Эталонные источники

1. **Material Design (Google)**
   - H1: line-height 1.0-1.2, letter-spacing -0.02em
   - Body: line-height 1.5-1.75, letter-spacing 0.01-0.02em

2. **Human Interface Guidelines (Apple)**
   - Large Display: line-height 1.0-1.1, letter-spacing -0.01em
   - Body: line-height 1.4-1.6, letter-spacing 0.01em

3. **Type Scale (Modern Web)**
   - Для Inter: рекомендуется line-height 1.0-1.2 для заголовков
   - Body: 1.5-1.7 для оптимальной читаемости

4. **Surf Coffee (эталон)**
   - Очень большие заголовки: плотный line-height, минимальный letter-spacing
   - Чистый, минималистичный подход

## Применённые значения

### CSS переменные

```css
/* Letter-spacing */
--letter-spacing-h1: -0.02em;        /* -1.8px для 90px */
--letter-spacing-h2: -0.01em;        /* -0.56px для 56px */
--letter-spacing-h3: 0em;
--letter-spacing-h4: 0.01em;         /* 0.24px для 24px */
--letter-spacing-body: 0.02em;       /* 0.32px для 16px */
--letter-spacing-label: 0.05em;      /* для меток и кнопок */
--letter-spacing-logo: 0.08em;       /* для логотипа */

/* Line-height */
--line-height-h1: 1.0;               /* очень плотный */
--line-height-h2: 1.1;
--line-height-h3: 1.2;
--line-height-h4: 1.3;
--line-height-body: 1.6;             /* комфортное чтение */
--line-height-body-large: 1.65;
--line-height-label: 1.4;
--line-height-button: 1.5;
```
